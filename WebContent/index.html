J2EE Demonstrations

JSP examples The JSP (Java Server Page) pages are indicated by the suffix JSP. When one of them is called, the JSP processor writes a servlet for the JSP and then registers the servlet with the application container. The location of the TLD (Tag Library Definition) file is given by the <jsp-config><taglib> tag in the WEB-INF/web.xml file.

JSF examples The JSF (Java Server Faces or Java Server Facelets) xhtml files are indicated by mapping the URL values to the javax.servlet.faces.FacesServlet class. JSF also requires a context listener class to be specified. For MyFaces, this is org.apache.myfaces.webapp.StartupServletContextListener.

Servlet examples - The mapping of the URL's to the servlets is given by WEB-INF/web.xml.

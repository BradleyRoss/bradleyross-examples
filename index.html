Project bradleyross-examples

The purpose of this project is to demonstrate a number of examples and demonstrations for Java applications and Application Servers. The documentation (this page) is located at http://bradleyross.github.io/bradleyross-examples while the project itself is at http://github.com/Bradleyross/bradleyross-examples.

These are a number of programs that I have developed over the years, and I am reviewing them as I place them in this collection to make sure that I have the correct jar files and that they will work in Tomcat without warnings or error messages. However, I hope that you will find them useful in their current state.

  • Javadocs These are the Javadocs for the Java classes.
  • Some examples using JavaScript These pages were copied from the bradleyross-examples project so tht they can be seen without having to load the project into an application server.
  • Sample Database This is the SQL code used to create the sample database. I have used this code to create tables in both Postgres and MySQL.
  • Using Git These are some notes on using Git. This documentation and the accompanying code are both stored on http://www.github.com.
  • http://bradleyaross.wordpress.com This is the location of my blog.

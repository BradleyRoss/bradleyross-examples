The packages starting with local and bradleyross are part of a general purpose software library that I had previously developed for educational purposes and to support other projects that I worked on. These packages are freely distributable.

Information on the Java core classes can be found at the following sites.

  • http://docs.oracle.com/javase/7/docs/api Java™2 Platform Standard Edition 7 API Specification

  • http://docs.oracle.com/javaee/7/api/ Java™ Platform Enterprise Edition, v 7 API Specifications

  • http://javamail.java.net/nonav/docs/api JavaMail™ API documentation

  • http://www.slf4j.org/api/" slf4j

The following classes are used with the log4j logging library: {@link org.apache.log4j.rolling.RollingFileAppender}, {@link org.apache.log4j.rolling.TimeBasedRollingPolicy}, and {@link org.apache.log4j.PatternLayout}.

There appears to be a problem with Microsoft Internet Explorer™ if the class files for the applets are in JAR files. For this reason, I moved the Java code for the applets to the top level of the directory structure.
